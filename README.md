# react-native-bluebreeze

BlueBreeze bindings for React Native. A modern Bluetooth LE library.

## Installation

```sh
npm install react-native-bluebreeze
```

## Usage


```js
import { multiply } from 'react-native-bluebreeze';

// ...

const result = multiply(3, 7);
```
